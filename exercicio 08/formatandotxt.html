Principais formatação

Negrito/Destaque

Negrito: Para deixar a frase em negrito, era muito comum se usar a tag Bolt que consistia em colocar a frase entre Bs (<,b,>).

Hoje com a chegada das tags semanticas, temos que por a frase desejada entre <,strong,> que inglês é sinônimo de Força, poder. Gerando assim significado a tag

Itálico/Ênfase

Antigamente para dar ênfase a uma determinada frase ou deixar as frases de maneira italica era usada a tag <,i,>

Atualmente a maneira recomentadada de se dar ênfase a uma frase é com a tag <,em,>

Marcador de texto

Para marcar texto é utilizado a tag <,Mark,>

Textos grandes e pequenos

Para aumentar o tamanho das letras é usado CSS, mas não foi assim pra sempre. Antigamente para escrever frases com letras um pouco maiores, era se usado a tag <,big,>. Mas acabou ficando obsoleta segundo a nova norma do HTML5 por se tratar de uma tag que somente deixa as letras maiores, mas sem nenhum significado,. Diferentemente da tag que faz exatamente o contrário dela.

Para frases com letras menores era, e ainda é se usado a tg <,small,>

Deletando textos

Para expressar que alguma parte to texto deve ser lida, mas não considerada existe o comando <,del,> texto deletado

Incuindo textos

Após deletar o texto, para adiocionar o texto modificado e dar ênfase, é usado a tag <,ins,> texto inserido

Textos flutuando acima

Para elevar um caractérie, ou uma palavra. É comum se usar a tag <,sup,>

Por exemplo :22 = 4 => 22 = 4

Textos flutuando abaixo

Para elevar um caractérie, ou uma palavra. É comum se usar a tag <,sub,>

Por exemplo :H2O => H2O
